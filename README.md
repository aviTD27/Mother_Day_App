# 🌸 Mother's Day Greeting App

A simple and heartfelt React application created to celebrate Mother's Day.

This project displays a personalized message dedicated to my mom, written with love and appreciation.
The layout is clean, centered, and styled with soft colors to match the theme of the day.

## 💐 Message

> "Dear Mom,
> Thank you for your love, strength, patience, and everything you’ve done for me.
> You are my inspiration and my heart.
>
> — With love, Avi 💐"
